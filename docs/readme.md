# SmartMortgage Calculator

An advanced mortgage amortization and savings calculator that visualizes the long-term benefits of extra principal payments using real-time dashboards and AI-driven financial insights.

## Features

- **Dual Payment Strategies**: Support for both one-time lump sum and recurring monthly extra payments.
- **Dynamic Savings Dashboard**: Real-time visualization of total interest saved and years removed from the mortgage.
- **Granular Breakdown**: Displays savings broken down by daily, weekly, monthly, and yearly impact.
- **AI Financial Advisor**: Integrated Google Gemini API to provide personalized insights on wealth-building strategies.
- **Multi-Currency Support**: Support for major global currencies (USD, EUR, GBP, JPY, etc.).
- **Responsive Design**: Fully optimized for mobile and desktop viewing.

## Tech Stack

- **Frontend**: React (v19)
- **Language**: TypeScript
- **Styling**: Tailwind CSS
- **Icons**: Lucide React
- **AI**: Google Generative AI (@google/genai)
- **Charts**: Recharts (for visualization components)

## Getting Started

1. Set your `API_KEY` environment variable for Gemini AI insights.
2. Open `index.html` in a supported browser.
